Installing ShelfLine & Setup

Complete Raspberry Pi + Android setup guide

How ShelfLine Works

ShelfLine allows Android devices to scan QR codes, capture expiry dates using OCR, and store records directly into a MariaDB database hosted on a Raspberry Pi.

Android phone
└─ Step 1: scan product QR / barcode
└─ Step 2: photograph expiry label (OCR → DDMMYYYY)
└─ Step 3: confirm quantity & remarks
└─ Save → Tailscale VPN → Raspberry Pi MariaDB

1. Raspberry Pi Setup

Install MariaDB on your Raspberry Pi and connect it securely using Tailscale.

sudo apt update && sudo apt upgrade -y sudo apt install mariadb-server -y sudo mariadb-secure-installation

Edit MariaDB config:

sudo nano /etc/mysql/mariadb.conf.d/50-server.cnf

Change:

bind-address = 0.0.0.0

2. Database Setup

Create the ShelfLine database and scanning table.

CREATE DATABASE qr_scanner; USE qr_scanner; CREATE TABLE scan_records ( id INT AUTO_INCREMENT PRIMARY KEY, qr_code VARCHAR(500) NOT NULL, expiry_date CHAR(8) NOT NULL, quantity INT NOT NULL DEFAULT 1, remarks VARCHAR(500) NOT NULL DEFAULT '', scanned_at DATETIME DEFAULT CURRENT_TIMESTAMP );

Enable access for the app:

CREATE USER 'scanner_app'@'%' IDENTIFIED BY 'YourPassword'; GRANT ALL PRIVILEGES ON qr_scanner.* TO 'scanner_app'@'%';

3. Android App Setup

Download the APK and connect it to your Raspberry Pi database server.

Download ShelfLine APK →
  1. Open ShelfLine
  2. Tap the menu icon
  3. Enter your Pi Tailscale IP
  4. Enter database password
  5. Save settings

Settings are stored locally and persist between sessions.

Building ShelfLine from Source

ShelfLine is built using Flutter and supports Android devices running Android 8+.

# Install dependencies flutter pub get # Run debug build flutter run # Build release APK flutter build apk --release

Release APK output: build/app/outputs/flutter-apk/app-release.apk
